## Getting Started

### Build from source

Create a debian binary

```sh
git clone https://github.com/soufrabi/chatbot-client.git
cd chatbot-client
./scripts/linux.sh setup
./scripts/linux.sh build-deb
cd build_binary
sudo apt install ./chatbot-client.deb
```
